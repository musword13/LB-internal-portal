# LINE Bank Internal Portal

銀行內部員工整合平台，以單一介面取代多套難用的廠商系統（如 VITALSESP ESP），結合 AI 智慧功能提升作業效率。

## 核心功能

### 1. 合約文件歸檔 + 財務付款系統
- 上傳合約 PDF，AI 自動解析關鍵欄位（簽約對象、金額、期間、付款條件等）
- 解析結果可即時編輯修正，顯示 AI 信心度
- 一鍵確認歸檔並自動建立分期付款草稿
- 付款排程管理，到期自動提醒

### 2. 簽核流程 + Teams / Slack 通知
- 視覺化多階段簽核流程
- 核准 / 退回後自動透過 Microsoft Teams 或 Slack 發送個別通知
- 支援付款、IT 需求、請假、合約續約等多種簽核類型

### 3. IT 需求申請 + JIRA 整合
- Kanban 看板追蹤所有 IT 需求狀態
- 核准後自動建立 JIRA Ticket，可指定 Project 與開發團隊
- 影響系統多選欄位（台幣核心、外幣核心、信用卡系統、簽帳金融卡系統等）
- 自動透過 Teams / Slack 通知相關人員

### 4. 內規管理辦法
- 各部門內規上傳與歸檔管理（15+ 部現行法規）
- **AI 語義搜尋**：自然語言查詢，自動定位相關條文並提供摘要（如「差旅費住宿上限」→ 差旅費報支要點 第6條）
- 分部門 / 分類別篩選，版本追蹤
- AI 自動建立全文索引與關聯性分析

### 5. 智慧工單（意圖驅動）
- 自然語言輸入，AI 自動識別意圖（請假、IT 報修、採購、會議室、付款、內規查詢）
- 動態產生對應表單欄位，一鍵送出
- 支援範例快速填入

### 6. 企業公告欄 + AI 公告助手
- 分級公告（緊急 / 全行 / 人事 / 教育訓練 / 行政）
- **AI 公告草稿產生器**：輸入簡短描述，自動產生正式公告
- **AI 潤稿**：一鍵改善公告文字
- **AI 翻譯**：中英文互譯
- **AI 合規檢查**：自動偵測公告是否涉及敏感資訊
- 預覽 / 儲存草稿 / 即時發布

## 技術架構

- **前端**：純 HTML / CSS / JavaScript（單一檔案，無框架依賴）
- **後端**：Node.js + Express 靜態伺服器
- **整合服務**：Microsoft Teams、Slack、JIRA、Google Calendar、AI 文件解析引擎（Claude API）
- **品牌色**：LINE Green `#06C755` + LINE Bank Blue `#2E8BFF`
